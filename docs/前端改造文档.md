# 音乐榜单可视化平台 — 前端改造文档

## 项目概况

- **框架**：Vue 2 + ECharts 4 + Highcharts 8 + @jiaminghi/data-view
- **改造目标**：将原汽车大数据大屏替换为音乐榜单可视化大屏
- **后端接口前缀**：`/music/`（原为 `/cars/`）

---

## 当前布局与组件对照

```
┌─────────────┬─────────────┬──────────────────────┬──────────────┬──────────────┐
│ centerLeft1 │ centerLeft2 │       center         │ centerRight2 │ centerRight1 │
│   ①饼图     │   ②词云     │ ③数字卡+④排行+⑤三环 │ ⑥区间+⑦留榜│   留榜表    │
├─────────────────────────────┼────────────────────────────────────────────────────┤
│      bottomRight（左位置）  │              bottomLeft（右位置）                  │
│      ⑧今日实时榜单          │              ⑨历年演化3D柱状图                     │
└─────────────────────────────┴────────────────────────────────────────────────────┘

注意：底部两个组件的名称与位置是反的，bottomRight 显示在左边，bottomLeft 显示在右边
```

```vue
    ></Echart>

import Echart from '@/common/echart';
export default {
  data() {
    return {
      options: {}
    };
  },
  components: { Echart },
  props: {
    cdata: {
      type: Array,
      default: () => []
    }
  },
  watch: {
    cdata: {
      handler(newData) {
        if (!newData || newData.length === 0) return
        this.options = {
          backgroundColor: 'transparent',
          series: [{
            type: 'wordCloud',
            shape: 'circle',
            left: 'center',
            top: 'center',
            width: '90%',
            height: '90%',
            right: null,
            bottom: null,
            sizeRange: [14, 60],         // 字体大小范围
            rotationRange: [-45, 45],    // 旋转角度范围
            rotationStep: 15,
            gridSize: 8,
            drawOutOfBound: false,
            textStyle: {
              fontFamily: 'sans-serif',
              fontWeight: 'bold',
              // 颜色随机从大屏主题色中取
              color: function () {
                const colors = [
                  '#37a2da', '#32c5e9', '#9fe6b8',
                  '#ffdb5c', '#ff9f7f', '#fb7293',
                  '#e7bcf3', '#8378ea', '#00bcd4'
                ]
                return colors[Math.floor(Math.random() * colors.length)]
              }
            },
            emphasis: {
              focus: 'self',
              textStyle: { shadowBlur: 10, shadowColor: '#333' }
            },
            data: newData   // [{name, value}] 格式，直接使用
          }]
        }
      },
      immediate: true,
      deep: true
    }
  }
};
</script>
```

## 第十四步：完整替换 `src/components/highcharts/index.vue`

原来是汽车品牌3D柱状图（静态数据），改为**歌手历年上榜演化3D柱状图**（动态数据）：

后端返回格式（`music/bottomRight/`）：
```
{
  "years": [2018, 2019, ..., 2025],
  "singers": ["周深", "薛之谦", ...],
  "series": [
    {"singer": "周深", "data": [3, 8, 12, 20, 25, 30, 35, 23]},
    ...
  ]
}
```

```vue
<template>
  <div id="app">
    <div ref="chartContainer" style="width:100%;height:460px;"></div>
  </div>
</template>

<script>
import Highcharts from 'highcharts';
import Highcharts3D from 'highcharts/highcharts-3d';
Highcharts3D(Highcharts);

export default {
  name: 'SingerEvolutionChart',
  data() {
    return {
      chart: null
    }
  },
  async mounted() {
    // 先用空配置初始化图表
    this.chart = Highcharts.chart(this.$refs.chartContainer, {
      chart: {
        type: 'column',
        options3d: {
          enabled: true,
          alpha: 10,
          beta: 30,
          depth: 100,
          viewDistance: 25,
        },
        backgroundColor: 'rgba(19, 25, 47, 0.6)'
      },
      title: {
        text: '歌手历年上榜次数演化',
        style: { color: '#ffffff' }
      },
      xAxis: {
        categories: [],   // 年份，后续填入
        labels: { style: { color: '#ffffff' } }
      },
      yAxis: {
        title: { text: '上榜周次', style: { color: '#ffffff' } },
        labels: { style: { color: '#ffffff' } }
      },
      legend: {
        enabled: true,
        itemStyle: { color: '#ffffff' }
      },
      tooltip: {
        headerFormat: '<b>{point.key}年</b><br>',
        pointFormat: '{series.name}: <b>{point.y}</b> 周'
      },
      plotOptions: {
        column: {
          depth: 25,
          groupZPadding: 10
        }
      },
      series: []
    })

    // 请求历年演化数据
    const res = await this.$request.get("music/bottomRight/")
    const data = res.data

    // 更新 X 轴为年份
    this.chart.xAxis[0].setCategories(data.years.map(String))

    // 逐个添加歌手 series（每个歌手一个系列）
    data.series.forEach(s => {
      this.chart.addSeries({
        name: s.singer,
        data: s.data
      }, false)   // false = 不立即 redraw，等全部加完再 redraw
    })

    this.chart.redraw()
  },
  beforeDestroy() {
    if (this.chart) {
      this.chart.destroy()
      this.chart = null
    }
  }
}
</script>
```
